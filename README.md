## Installation and Compilation

### Dependencies

 - Docker
 - Docker-Compose

### Run-time, baby!

Configure files:
 - bot.env
 - backup.env

Using `Docker` and `Docker-Compose` to build container images and run:    
```
$ docker-compose build
$ docker-compose up
```
