## Installation and Compilation

### Dependencies

Run-time:
 - Docker
 - Docker-Compose
 
 Compile-time:
 - Maven

### Doing it

Configure files:
 - bot.env
 - backup.env

Using `Docker` and `Docker-Compose` to build container images and run:    
```
$ docker-compose build
$ docker-compose up
```
